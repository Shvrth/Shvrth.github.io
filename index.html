AnimalHaze3K Dataset

A specialized image dehazing dataset focusing on animal subjects in diverse environmental conditions. Contains 4,636 carefully curated images across multiple splits.

Dataset Splits

  • Training Set:
    1,041 clean reference images
    3,123 hazy counterparts
  • Test Set:
    59 clean images
    177 hazy versions
  • Validation Set:
    59 clean images
    177 hazy versions
