AnimalHaze3K Dataset

Hugging Face Download Dataset

AnimalHaze3k is an image dehazing dataset focused on animal subjects in diverse environments. Contains 4,636 carefully curated images across three splits:

  • Training Set:
    1,041 clean reference images
    3,123 hazy counterparts
  • Test Set:
    59 clean images
    177 hazy versions
  • Validation Set:
    59 clean images
    177 hazy versions
